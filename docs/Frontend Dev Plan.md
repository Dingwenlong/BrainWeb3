# 前端“大脑神经活跃可视化”设计开发计划

## 状态说明
- `已完成`：仓库中已有对应页面、接口接线或可演示交互。
- `进行中`：已有占位实现或样板链路，但还未达到目标形态。
- `未开始`：文档已规划，但仓库中尚未落地。

## 摘要
- 目标是在 `数据详情页` 做一个基于真实 EEG 数据驱动的 `3D 大脑热力图`，让用户能直观看到脑区随时间变化的活跃程度。
- 首版采用 `频段功率` 作为活跃度指标，不把原始脑电波形直接暴露给前端；前端消费的是后端预计算后的脑区强度序列。
- 交互范围固定为 `3D 热力图 + 时间轴回放`，不做首版多任务对比或复杂分析工作台。

## 当前落地情况（截至 2026-04-20）
- `已完成`：数据详情页已接入 `brain-activity` 接口、频段切换、时间轴拖动、播放暂停、当前帧脑区强度列表。
- `已完成`：总览页与详情页已打通上传 -> 数据详情 -> 存证摘要 -> 运行态展示链路。
- `进行中`：`BrainActivityViewer` 已有“神经舱”可视化占位组件，但仍不是 `Three.js` 真 3D 脑模型热力图。
- `未开始`：真实 8 分区脑模型、脑区悬停浮层、按时间窗增量加载、前端专项测试与权限校验。

## 关键实现
### 视觉与交互
- `已完成` 视觉方向已落为“医疗科幻神经舱”：深色背景、冷蓝/青绿主光、局部橙黄强调，不是默认 Vite 风格。
- `进行中` 页面主体已基本形成三块：可视化区、时间轴回放区、当前帧指标区；其中可视化区仍为占位实现。
- `未开始` 3D 区域拆分为 `左额叶、右额叶、左顶叶、右顶叶、左颞叶、右颞叶、枕叶、中央运动区` 8 个可着色区域。
- `进行中` 已支持播放、暂停、拖动时间轴、查看当前时间窗活跃值；脑区悬停浮层尚未完成。
- `已完成` 默认展示单一频段并支持 `Delta/Theta/Alpha/Beta/Gamma` 切换；未做多频段并排对比。

### 数据链路与计算方式
- `进行中` Python EEG 服务已提供 `EDF/GDF/BDF` 样板接口与 `brain-activity` mock/样例数据返回；真实 MNE 预处理仍未接入。
- `进行中` 当前已按 `EEG 通道组 -> 脑区` 聚合思路返回区域强度，但仍属于样板数据链路，不是真实源定位或真实脑区映射结果。
- `进行中` 接口中已有脑区、电极组、时间窗和强度结构；lookup 表与真实 10-20/64 导映射仍需补齐。
- `已完成` 后端当前只向前端返回 `时间窗 + 脑区强度 + 频段信息 + 数据质量标记`，不下发原始 EEG 明文。
- `未开始` 数据详情页“先摘要后增量拉取时序”的分段加载机制，当前仍是一次性加载该数据集的时间窗序列。

## 接口与类型
- `已完成` 已有聚合接口：`GET /datasets/{id}/brain-activity`
- `进行中` 当前稳定支持的查询参数以 `band` 为主，`windowSize`、`stepSize`、`timeStart`、`timeEnd` 仍未完整开放到现有前后端链路。
- `已完成` 返回结构当前已包含：
  - `datasetId`
  - `samplingRate`
  - `regions`
  - `frames`
  - `qualityFlags`
  - `generatedAt`
- `进行中` 关键类型已部分落地为：
  - `BrainRegion`: 脑区枚举与显示名称
  - `ActivityFrame`: 单个时间窗的脑区活跃度快照
- `未开始` `RegionChannelMapping` 独立类型沉淀。
- `未开始` `BandPowerSeries` 独立类型沉淀。
- `进行中` 前端组件结构当前状态：
  - `已完成` `BrainActivityViewer`: 页面主容器已存在。
  - `未开始` `Brain3DHeatmap`: 真实 Three.js 脑模型与区域着色尚未落地。
  - `已完成` `ActivityTimeline`: 时间轴与播放控制已合并进详情页实现。
  - `已完成` `RegionMetricsPanel`: 当前帧和脑区指标清单已在详情页中可用。

## 测试与验收
- `未开始` 映射测试：给定一组已知电极 band power，稳定得到正确脑区聚合结果。
- `进行中` 接口测试：后端已有基础接口单测，但 `brain-activity` 的频段/时间范围/坏数据专项覆盖仍不足。
- `未开始` 前端专项测试：时间轴拖动后 3D 热力图同步刷新；切换频段后颜色和指标同步变化。
- `未开始` 性能测试：5 分钟级 EEG 数据详情页加载与回放性能验证。
- `未开始` 安全测试：无权限用户不可获取可视化摘要；目前还未接入统一鉴权。
- `进行中` 验收标准已完成一半：用户已能打开 EEG 数据详情、查看时间变化和频段切换；但 `3D 大脑热力图` 与脑区悬停说明尚未完成。

## 假设与默认
- `已完成` 前端技术沿用 `Vue 3`；3D 渲染目标仍默认使用 `Three.js`。
- 数据来源默认使用 `PhysioNet EEG Motor Movement/Imagery Dataset` 的 64 导数据做首版验证。
- 首版“活跃度”默认使用标准化后的 band power 强度，不直接解释为医学诊断结论。
- 若暂时拿不到高质量解剖级脑模型，首版允许使用简化分区脑模型，但区域划分和时间轴交互不变。
- `已完成` 可视化目前作为数据详情页子模块存在，没有拆成独立大屏或分析工作台。
