## 4. 模拟正式环境说明

若当前是在“模拟正式环境验收”口径下：

- 可以保留这些账号用于 smoke 和验收演练
- 但不应再使用默认密码语义对外宣称“正式环境已就绪”

更推荐的做法是：

1. 保留当前 `actorId / role / organization`
2. 将密码替换成单独分发的联调密码
3. 在验收报告中注明“账号结构沿用演示账户，密码已脱离 demo 默认值”
